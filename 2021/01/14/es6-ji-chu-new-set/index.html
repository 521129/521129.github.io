Es6 基础之 new set

Set对象存储的值总是唯一的
Set对象方法

方法 描述
add 添加某个值，返回Set对象本身
clear 删除所有的键/值对，没有返回结果
delete 删除某个键，返回true、如果删除失败，返回false
forEach 对每个元素执行指定操作
has 返回一个布尔值，表示某个键是否在当前Set对象之中

Set对象作用

  • 数组去重
  • 并集
  • 交集
  • 差集
